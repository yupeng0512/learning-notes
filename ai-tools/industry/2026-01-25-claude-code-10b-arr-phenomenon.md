## 核心洞见

### 1. 「副业」到「10亿美元」的启示

Claude Code 不是大厂立项、不是融资砸钱，而是工程师的「副业」起步。

**关键因素**：
- 解决真实痛点（不是玩具）
- 定位精准（AI 同事 vs 补全工具）
- 与真实工作流深度融合

### 2. 「100% AI 写代码」的里程碑

Boris 作为 Claude Code 开发负责人，自己 100% 用 AI 写代码——这不是营销话术，而是产品信心的最强背书。

### 3. 收购 Bun 的战略意义

> AI 编码工具已进入中后台基础设施时代

收购 Bun 表明 Anthropic 在构建完整的开发者生态，而非单点工具。
